
# Uploads Directory

This directory contains uploaded images for the HotelsSifnos website.

## Structure

- `/hotels/` - Hotel images
- `/beaches/` - Beach images 
- `/rooms/` - Hotel room images
- `/banners/` - Banner images for various pages
- `/misc/` - Miscellaneous images

## Image Naming Convention

- Use lowercase letters
- Use hyphens instead of spaces
- Include descriptive names
- Add dimensions if relevant

Example: `elies-resort-pool-1200x800.jpg`
