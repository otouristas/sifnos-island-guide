
# Hotel Images

This directory contains all hotel-related images:
- Main hotel photos
- Room photos
- Hotel logos
- Amenity icons

Structure:
- `HTL1/` - Images for hotel with ID HTL1
  - `main.jpg` - Main hotel photo
  - `logo.png` - Hotel logo
  - `rooms/` - Room photos
  - `amenities/` - Amenity photos
